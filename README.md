# 💁 Service Introduce 
## O2O(Online to Offline) 기반 스마트 카페 시스템 (Android)
- 앱을 통해 주문 및 결제 가능
- 심플하고 직관적인 UI
- 사용자, 관리자 용도에 맞는 앱 제공 (관리자 앱은 태블릿에서 사용 가능)

<br>

###  개발환경
- Android Studio @Arctic Fox
- Kotlin
- MySQL @8.0.26
- Spring boot
- Git, GitLab

> Galaxy S10 (사용자앱), Galaxy Tab S7 (관리자앱) 에 최적화되어 있습니다.
